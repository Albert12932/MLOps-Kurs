## Название датасета
Emotion Text Dataset

## Происхождение данных
Датасет представляет собой набор текстов с аннотациями эмоций.  
Каждому тексту соответствует набор бинарных меток, отражающих наличие определённых эмоций.

Датасет используется исключительно в учебных и исследовательских целях.

## Структура данных
Каждая строка содержит:
- текст
- набор колонок, соответствующих эмоциям (0 или 1)

Примеры эмоций:
- joy
- sadness
- anger
- fear
- love
- surprise
- optimism
- neutral  
и другие

## Препроцессинг данных
В проекте используются несколько версий датасета:

### Версия 1
- удаление пустых текстов
- фильтрация неясных примеров
- базовая очистка текста

### Версия 2
- дополнительная очистка
- ограничение размера датасета (30k примеров)
- используется для ускорения экспериментов

### Версия 3
- альтернативная очистка текста
- лемматизация
- используется для сравнения влияния препроцессинга

Все версии датасета сохраняются в папке `data/preprocessed`.

## Разделение данных
В рамках экспериментов данные разделяются на:
- обучающую выборку
- валидационную выборку

Разделение производится внутри обучающих скриптов.


## Ограничения датасета
- Возможны ошибки разметки
- Некоторые эмоции представлены реже других
- Тексты могут содержать неоднозначные эмоциональные сигналы
