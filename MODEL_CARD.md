## Название модели
Emotion Detection Model

## Назначение
Модель предназначена для определения эмоциональной окраски текста.  
На вход подаётся текст, на выходе модель возвращает вероятности для набора эмоций и список наиболее вероятных эмоций.


## Архитектуры моделей
В проекте используются несколько подходов:

1. Baseline модель  
   - TF-IDF векторайзер  
   - Логистическая регрессия  
   Используется как простая базовая линия для сравнения.

2. ModernBERT (frozen)  
   - Предобученная модель modernbert-base-go-emotions  
   - Замороженные веса энкодера  
   - Обучается только классификатор

3. ModernBERT (fine-tuned)  
   - Та же архитектура  
   - Размораживание части или всех слоёв  
   - Полное дообучение под задачу

## Данные
Для обучения используются тексты из датасета эмоций (см. DATASET_CARD.md).  
Задача формулируется как многоклассовая классификация.

## Метрики качества
Для оценки качества моделей используются:
- F1 micro
- F1 macro

Baseline модель показывает более низкие значения метрик и используется как точка отсчёта.  
Модели на основе ModernBERT демонстрируют существенно лучшие результаты.

## Эксперименты
Все эксперименты логируются с помощью MLflow:
- параметры обучения
- используемая версия датасета
- итоговые метрики
- артефакты моделей

Это позволяет воспроизводить результаты и сравнивать разные подходы.

## Ограничения
- Модель обучалась на текстах на английском языке
- Возможны ошибки на коротких или неоднозначных текстах

## Хранение моделей
Веса моделей не хранятся в репозитории из-за ограничений по размеру.  
Все обученные модели сохраняются как артефакты в MLflow.
