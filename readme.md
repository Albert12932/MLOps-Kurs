Emotion MLOps

Проект представляет собой MLOps-пайплайн для классификации эмоций в текстовых сообщениях.  
Реализован процесс обучения модели, логирование экспериментов в MLflow и развертывание REST API для инференса через FastAPI.

FastAPI - REST API для предсказаний
MLflow - трекинг экспериментов, метрик и моделей
scikit-learn - обучение модели логистической регрессии
Poetry - управление зависимостями
Docker/Docker Compose - контейнеризация и запуск сервисов
